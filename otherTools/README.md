**"delete_json"**
- This script is used to delete matching .json, .png, .atlas, and atlas.txt files with the same name simultaneously.

**"delete_skel"**
- This script is used to delete matching .skel, .png, .atlas, and atlas.txt files with the same name simultaneously.

**"rename_spine"**
- Rename the .spine file to match the folder name.

**"export_spine"**
- This script is used to call spine.exe and export JSON files via command line.
- The Spine version depends on the currently available version on your PC.
- If you want to use custom settings, you need to create your own json_settings file using Spine 2D at exporting window.
